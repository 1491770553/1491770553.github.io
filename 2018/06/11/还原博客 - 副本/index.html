我的兄弟叫顺溜之还原博客

LiuJunWei

2018-06-11

糟糕，重装系统了，博客没了怎么办！

大概分为下面几个步骤

偷偷告诉你们


没有备份博客也搞不定鸭

第一步：环境搭建

安装git （因为要给github上传文章）
  1. Windows: 直接下载和安装
  2. Mac：进入git网站下载和安装，或者进入这里下载
  3. Ubuntu：打开终端复制粘贴命令 sudo apt-get install git-core

安装Node.js（因为Hexo是基于Node.js开发的）

进入官网: 点我
下载安装包，直接点击安装就可以了

第二步：安装Hexo和安装deploy-git（也就是部署的命令,这样你才能用命令部署到GitHub）

以下命令都是直接复制粘贴到电脑终点运行的（也就是超级管理员）

上面的环境配置成功以后就可以使用npm安装Hexo了

这个时候需要先安装deploy-git ，也就是部署的命令,这样你才能用命令部署到GitHub。

然后依次执行下面的命令



  1. 其中 hexo clean清除了你之前生成的东西，也可以不加。
  2. hexo generate 顾名思义，生成静态文章，可以用 hexo g缩写
  3. hexo deploy 部署文章，可以用hexo d缩写
  4. 注意deploy时可能要你输入username和password。
  5. 差不多说明部署成功了，过一会儿就可以你的网址看到你的博客了！！

一般就这样就完成了
